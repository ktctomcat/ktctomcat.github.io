# TomoDesign Works

🌱 プロジェクト概要

TomoDesign Worksは、地域文化とWeb技術の融合を目指す実験的なデザイン・開発プロジェクトです。 公共性・アクセシビリティ・意味論的設計を重視し、持続可能で保守性の高いWeb制作を実践しています。

## 🎯 目的

地域文化や公共性を尊重した情報発信の仕組みづくり

意味論的なHTML/CSS設計によるアクセシビリティ最適化

UI/UXの細部までこだわるデザイン哲学の実装

GitHub Pages + Jekyll Minimalテーマを活用した軽量な運用

## ✨ 特徴

意味論的設計：HTMLタグの役割を明確化し、スクリーンリーダー対応を強化

CSS設計：Flexbox・margin・borderの適切な使い分けで保守性を確保

コメント整備：コードの可読性・再利用性を高めるための詳細なコメント

軽量運用：Jekyll + GitHub Pagesによる高速・低コストな公開
